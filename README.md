# User Management CRUD

## 🚀 Tech Stack
- Angular
- Node.js
- Express
- MongoDB
- Angular Material

## ✨ Features
- Add Employee
- Update Employee
- Delete Employee
- View Employees

## 🛠 Setup Instructions

### Backend
cd Backend
npm install
npm start

### Frontend
cd frontend
npm install
ng serve
